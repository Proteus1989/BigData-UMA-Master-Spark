 ##### IntelliJ
 - Create a new maven project (File / New / Project... / Maven)
 - Set GroupId and ArtefactId as ``org.masterbigdata.spark``
 - Set project name as ``Spark``
 - Enable ``Enable auto-import``
 - Replace ``pom.xml`` and ``src`` into new project folder
